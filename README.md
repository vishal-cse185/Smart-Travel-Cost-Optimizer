# ✈️ Smart Travel Cost Optimizer

This project uses Dynamic Programming to find the minimum travel cost between cities with a constraint on maximum stops.

## 🔧 Technologies Used
- Python
- Dynamic Programming
- Streamlit (UI)

## 🚀 Features
- Finds cheapest route
- Supports stop constraints
- Real-world travel optimization

## 📥 Input Example
Cities: 4  
Flights:  
0 1 100  
1 2 100  
0 2 500  
2 3 100  

Source: 0  
Destination: 3  
Max Stops: 2  

## 📤 Output
Minimum Travel Cost: 300

## 🧠 Concept Used
Dynamic Programming (Bellman-Ford approach)
